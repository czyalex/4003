## CPIC pharmacogenomics highlights (gene / variant / alleles)
- **SLCO1B1 c.521T>C (rs4149056)** — decreased OATP1B1 function → ↑ simvastatin exposure and ↑ myopathy risk  
  - **Normal function (TT)** → standard dosing  
  - **Decreased function (TC)** → consider **lower dose** or **alternative statin** (e.g., pravastatin, fluvastatin); monitor for muscle symptoms/CK  
  - **Poor function (CC)** → **avoid simvastatin** or use only at low dose with close monitoring; prefer alternative statin
